# Lempel-Ziv ComplexityLZC Analysis Pipeline

This repository contains the full Python implementation used in the project:
- General info acquisition from (.mat)
- MATLAB (.mat) to NumPy (.npy) conversion
- Channel Selection
- Temporal segmentation
- LFP LZC Analyser 

## Type of files
– Visual Studio code notebooks
– output files in (.xlsx) format
